# illusory0x0/vscode 

This project provides MoonBit language bindings for the Visual Studio Code Extension API, enabling developers to build VSCode extensions using MoonBit.

## Project Status 

This project is generated by **AI Agent**, test coverage is very low.
If you meet any issues, please feel free to open an issue or a PR.


## Type Translation Scheme 

- Object -> Structs in [vscode_structs.mbt](./types/vscode_structs.mbt)
- Enums -> Enums in [vscode_enums.mbt](./types/vscode_enums.mbt) (using binary complement for negative values)
- Provider interfaces -> Structs with function type fields in [vscode_providers.mbt](./types/vscode_providers.mbt)
- Namespaces -> Structs in [vscode_values.mbt](./types/vscode_values.mbt)
- Union types -> Phantom unions in [unions.mbt](./types/unions.mbt) (no compile-time type checking, use `coerce` for unsafe casts) 
- Abstract classes -> Structs in [vscode_abstracts.mbt](./types/vscode_abstracts.mbt) (no compile-time type checking, use `coerce` for unsafe casts)
- Function type with rest parameters -> FuncRest in [func_rest.mbt](./types/func_rest.mbt) 

fields respect the original camelCase naming convention.
